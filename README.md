## 1. 파이썬 개요와 특징

- 인터프리터 언어로, 직관적이고 문법이 간결합니다.
- 다양한 플랫폼에서 실행 가능하고, 방대한 커뮤니티를 갖추고 있습니다.
- 활용 분야: 웹 개발, 데이터 과학, 머신러닝, 자동화 등

---

## 2. 변수, 자료형, 연산자

- 변수는 타입 선언 없이 사용: `x = 10`
- 주요 자료형: `int`, `float`, `str`, `bool`, `list`, `dict` 등
- 연산자:
  - 산술: `+`, `-`, `*`, `/`, `//`, `%`, `**`
  - 비교: `==`, `!=`, `<`, `>`
  - 논리: `and`, `or`, `not`

---

## 3. 조건문과 반복문

- 조건문:
```python
if x > 0:
    print("양수")
else:
    print("음수 또는 0")
```

- 반복문:
```python
for i in range(5):
    print(i)
```
4. 함수와 람다 함수
- 함수 정의:

```python
def greet(name):
    return f"Hello, {name}"
```

- 람다 함수 (익명 함수):

```python
square = lambda x: x**2
print(square(3))  # 9
```
