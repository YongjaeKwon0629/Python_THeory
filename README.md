## 1. 파이썬 개요와 특징

### 🔹 1.1 파이썬의 역사
- 1991년 Guido van Rossum이 개발
- 철학: "읽기 쉽고, 쓰기 쉬운 코드"

### 🔹 1.2 파이썬의 특징
- 인터프리터 언어
- 동적 타이핑
- 들여쓰기 기반 문법
- 크로스 플랫폼 지원

### 🔹 1.3 파이썬의 활용 분야
- 데이터 과학, 웹 개발, 인공지능, 자동화, IoT, 교육 등

## 3. 조건문과 반복문

- 조건문:
```python
if x > 0:
    print("양수")
else:
    print("음수 또는 0")
```

- 반복문:
```python
for i in range(5):
    print(i)
```
